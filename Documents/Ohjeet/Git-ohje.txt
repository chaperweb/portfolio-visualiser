Ohjeet Gitin käyttöä varten:

Branchit
	-Projekti pääbranch
	-Bugit omiin brancheihin
	-Featuret omiin brancheihin 



Commitit
	-Tee commit aina branchiin, jota käytät devatessa, ei koskaan masteriin!
	-Jokaiseen committiin kokonainen kuvaus siitä mitä on tehty/lisätty
	-Linkitä agilefant-tiketti committiin

Pull request
	-Kun tiketti on valmis, tee pull request.
	-Joku toinen devaaja käy koodisi läpi ja kun se on todettu hyväksi jenkins prosessi hoitaa tiketin mergen masteriin.
	-Tämä prosesi vaatii vielä hiomista ja jutellaan siitä lisää.



Sourcetree

	-Ladataan osoitteesta https://www.sourcetreeapp.com/
	-Kopioi codenton git sourcetreehen
